# Web Development Course - Vercel Deployment

This project is ready to be deployed on [Vercel](https://vercel.com/).

## Steps to Deploy

1. **Push to GitHub**
   - Make sure your code is committed and pushed to your GitHub repository.
2. **Import to Vercel**
   - Go to [vercel.com/import](https://vercel.com/import) and select your GitHub repo.
   - Set the output/public directory to `.` (the root folder).
   - No build command is needed (static site).
3. **Configure Custom Domain (Optional)**
   - You can add a custom domain in the Vercel dashboard.

## Project Structure
- All HTML, CSS, JS, and assets are in the root and subfolders.
- No server-side code is required for deployment.

## Notes
- If you use PHP or server-side features, they will not run on Vercel (static hosting only).
- For dynamic features, consider using Vercel Serverless Functions or an external backend.

**Happy deploying!**
