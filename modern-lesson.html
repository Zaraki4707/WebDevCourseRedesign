HTML

What is HTML?

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating Web pages. It describes the structure of Web pages using markup. HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.

It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes, and other items. It can embed scripts written in languages such as JavaScript which affect the behavior of HTML web pages.

To read the living standard of HTML, visit the official specification.

The Structure of HTML Pages

HTML documents follow a standard structure that defines how content is organized. Every HTML page starts with a document type declaration and contains specific elements that browsers use to render the page correctly.

Key Points:

  • <!DOCTYPE html> - Declaration that tells the browser this is an HTML5 document
  • <html> - Root element that wraps all content on the page
  • lang attribute - Specifies the language of the document (e.g., "en" for English)
  • <head> - Contains meta-information like title, character set, and CSS links
  • <meta charset="UTF-8"> - Defines character encoding for proper text display
  • <title> - Sets the page title shown in browser tab
  • <body> - Contains all visible content displayed in the browser

Headings

HTML has six levels of headings, which use the elements <h1> through <h6>. Headings are used to represent the hierarchical structure of the content. The <h1> element represents the highest level of heading, while the <h6> element represents the lowest level.

Paragraphs

The <p> element is used to create a paragraph. Browsers automatically add some space (margin) before and after each <p> element. Text is easier to understand when it is split up into units of text, and paragraphs provide this structure.
