CSS Layout Techniques

Explore modern and classic ways to structure your web pages using Box Model, Floats, Flexbox, and Grid.

1. The Box Model

In the default box model, when you set the width or height of an element, you are specifying the width or height of its content; any padding, border, and margins are then added to that width.

We can adjust the box model behavior with its box-sizing property:

  • box-sizing: content-box (default): The height and width only apply to the content.
  • box-sizing: border-box: The height and width properties include content, padding, and border.

Using border-box is generally recommended as it makes layout calculations much more intuitive.

2. CSS Layouts

Layout techniques have evolved significantly. We'll explore the classic Float method, followed by modern standards like Flexbox and Grid.

2.1 Float

Floats are one of the oldest methods for laying out web pages. A float pulls an element to one side of its container, allowing the document flow (usually text) to wrap around it.

  • If you float multiple elements in the same direction, they will stack alongside one another.
  • Floats are still the primary way to have text wrap around an image.

HTML Example

HTML 
<img class="float-img" src="../SRC/IMAGES/Oran.jpg">
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit...
</p>

CSS 
.float-img {
  float: left;
  margin-right: 15px;

2.2 Flexbox

Flexbox (Flexible Box Layout) is a powerful layout model that makes it easy to align elements and distribute space within a container, even when their size is unknown or dynamic.

Applying display: flex to an element turns it into a flex container, and its direct children become flex items.

Main Axis and Cross Axis

Flexbox works with two axes: the Main Axis (determined by flex-direction) and the Cross Axis (perpendicular to the main axis).

Basic Flex Container

CSS 
.flex-container {
  display: flex;
  flex-direction: row; /* row, row-reverse, column, column-reverse */
  justify-content: center; /* along main axis */
  align-items: center; /* along cross axis */
}

Flex Wrap

            

            
The flex-wrap property specifies whether flex items should wrap or not.

            
            

                

                     CSS
                    
                

                
.container {
    display: flex;
    flex-wrap: wrap; /* nowrap | wrap | wrap-reverse */
}

            

        


        
        

            

                
                
Alignment Properties

            

            
            

                
 Key Properties

                
    
                    
  • justify-content: Aligns items along the main-axis (horizontal by default).
    • 
                    
  • align-items: Aligns items along the cross-axis (vertical by default).
    • 
                    
  • align-self: Allows the default alignment (or the one specified by align-items) to be overridden for individual flex items.
    • 
                

            


            

                

                     CSS Example
                    
                

                
.container {
    display: flex;
    justify-content: center; /* space-between, space-around, flex-start/end */
    align-items: center; /* flex-start, flex-end, stretch, baseline */
    height: 200px;
}

            

        


        
        

            

                
                
CSS Grid Layout

            

            
CSS Grid Layout offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.


            

                

                     CSS
                    
                

                
.grid-container {
    display: grid;
    grid-template-columns: 200px auto 100px;
    grid-template-rows: 100px 100px;
    gap: 10px;
}

The Position Property

            

            
The position property specifies the type of positioning method used for an element.

            
            

                
    
                    
  • Static: Default value. Normal flow.
    • 
                    
  • Relative: Positioned relative to its normal position.
    • 
                    
  • Fixed: Positioned relative to the viewport.
    • 
                    
  • Absolute: Positioned relative to the nearest positioned ancestor.
    • 
                    
  • Sticky: Positioned based on the user's scroll position.
    • 
                

            

        


        
        

            

                
                
Responsive Design & Media Queries

            

            
Responsive web design makes your web page look good on all devices. It uses HTML and CSS to resize, hide, shrink, or enlarge a website.


            

                

                     Media Query Example
                    
                

                
/* Desktop First */
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}

/* Mobile First (Recommended) */
@media (min-width: 768px) {
    .container {
        flex-direction: row;
    }
}
