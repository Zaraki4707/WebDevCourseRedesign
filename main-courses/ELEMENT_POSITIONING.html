Element Positioning

Master the art of placing elements exactly where you want them using CSS position properties.

The Position Property

The initial value of the position property is static. Positioning allows you to remove elements from the normal document flow, letting you place them arbitrarily on the screen or relative to other elements.

Fixed Positioning

position: fixed lets you position an element relative to the viewport. It stays in the same place even when the page is scrolled. 

.fixed-element {
  position: fixed;
  top: 20px;
  right: 20px;
  z-index: 1000;
}

Absolute Positioning

position: absolute positions an element relative to its closest positioned ancestor (an ancestor with position: relative, absolute, or fixed).

Relative Positioning

position: relative allows you to move an element from its original position without affecting the surrounding elements. It also acts as a container for absolutely positioned children. 

.parent { position: relative; }
.child { position: absolute; top: 0; }
