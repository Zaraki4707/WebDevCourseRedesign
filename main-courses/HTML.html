HyperText Markup Language (HTML)

What is HTML?

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating Web pages. It describes the structure of Web pages using markup. HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.

It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes, and other items. It can embed scripts written in languages such as JavaScript which affect the behavior of HTML web pages.

To read the living standard of HTML, visit the official specification.

I. The Structure of HTML Pages

HTML documents follow a standard structure that defines how content is organized. Every HTML page starts with a document type declaration and contains specific elements that browsers use to render the page correctly.

Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First Page</title>
</head>
<body>
  <h1>Welcome to HTML!</h1>
  <p>This is my first webpage.</p>
</body>
</html>
Try This Code

Key Points:

  • <!DOCTYPE html> - Declaration that tells the browser this is an HTML5 document
  • <html> - Root element that wraps all content on the page
  • lang attribute - Specifies the language of the document (e.g., "en" for English)
  • <head> - Contains meta-information like title, character set, and CSS links
  • <meta charset="UTF-8"> - Defines character encoding for proper text display
  • <title> - Sets the page title shown in browser tab
  • <body> - Contains all visible content displayed in the browser

II. Text - Headings

HTML has six levels of headings, which use the elements <h1> through <h6>. Headings are used to represent the hierarchical structure of the content. The <h1> element represents the highest level of heading, while the <h6> element represents the lowest level.

Heading Examples

<h1>This is a Level 1 Heading</h1>
<h2>This is a Level 2 Heading</h2>
<h3>This is a Level 3 Heading</h3>
<h4>This is a Level 4 Heading</h4>
<h5>This is a Level 5 Heading</h5>
<h6>This is a Level 6 Heading</h6>

II.2 Paragraphs

The <p> element is used to create a paragraph. Browsers automatically add some space (margin) before and after each <p> element. Text is easier to understand when it is split up into units of text, and paragraphs provide this structure.

Paragraph Example

<p>
  A paragraph consists of one or more sentences that form a 
  self-contained unit of discourse. The start of a paragraph 
  is indicated by a new line.
</p>

<p>
  Text is easier to understand when it is split up into units. 
  For example, a book may have chapters with subheadings, and 
  under each heading there will be one or more paragraphs.
</p>

II.3 Bold and Italic

The <b> element is used to represent text in a bold format. The <i> element is used to represent text in an italic format. The following example demonstrates how to use these elements.

Bold and Italic Example

<p>
  This is how we make a word appear <b>bold</b>.
</p>

<p>
  This is how we make a word appear <i>italic</i>.

II.4 Superscript and Subscript

The <sup> element is used to represent superscript text. The <sub> element is used to represent subscript text.

Superscript and Subscript Example

<p>
  On the 10<sup>th</sup> of February, you will have an exam.
</p>

<p>
  The formula for water is H<sub>2</sub>O.
</p>

III. Lists - Ordered Lists

The <ol> element is used to create an ordered list. Each item in the list is placed between an opening <li> tag and a closing </li> tag. (The li stands for list item.)

Ordered List Example

<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
  <li>PHP</li>
</ol>

The <ul> element is used to create an unordered list. Each item in the list is placed between an opening <li> tag and a closing </li> tag.

Unordered List Example

<ul>

IV. Links

Links are created using the <a> element. Users can click on anything between the opening <a> tag and the closing </a> tag. You specify which page you want to link to using the href attribute.

Link Example

<a href="https://ensia.edu.dz/">Visit ENSIA</a>

<p>

V. Images

The <img> element is used to add images to a web page. The src attribute tells the browser where to find the image file. The alt attribute provides alternative text that describes the image.

Image Example

<img src="logo.png" alt="Company Logo" width="200">

VI. Tables

The <table> element is used to create tables. Tables are defined using the <tr> element for rows, <th> for header cells, and <td> for data cells.

Table Example

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>Ahmed</td>
    <td>20</td>
    <td>Algiers</td>
  </tr>
  <tr>
    <td>Sara</td>
    <td>22</td>
    <td>Oran</td>
  </tr>
</table>

VII. Forms

The <form> element is used to create HTML forms for user input. Forms can contain various input elements like text fields, checkboxes, radio buttons, and submit buttons.

Form Example

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name"><br><br>
  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email"><br><br>
  
  <label for="message">Message:</label><br>
  <textarea id="message" name="message"></textarea><br><br>
  
  <input type="submit" value="Submit">
</form>
